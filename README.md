# 2020-ci il üzrə Satış Performansı - Pet Store Layihəsi
Bu layihə, 2020-ci il üçün ev heyvanları mağazasının satış datalarının Google Sheets vasitəsilə analizini və vizuallaşdırılmasını əhatə edir. Layihənin məqsədi xam məlumatlardan istifadə edərək ən çox gəlir gətirən və tələb olunan məhsul kateqoriyalarını müəyyən etməkdir.

### Layihənin icra mərhələləri:
**Məlumatların təmizlənməsi və strukturlaşdırılması:** İlkin satış məlumatları düzgün kateqoriyalara ayrılaraq pivot cədvəllər vasitəsilə ümumiləşdirilib.
* **Vizuallaşdırma:** Satış trendlərini daha aydın izləmək üçün "Stacked Column Chart" (yığılmış sütunlu diaqram) qurulub. Bu qrafik, müxtəlif heyvan növləri üzrə məhsul qruplarının payını vizual şəkildə müqayisə etməyə imkan verir.
* **Şərti formatlaşdırma:** Satış həcmlərindəki fərqləri vizual olaraq vurğulamaq üçün "Conditional Formatting" tətbiq edilib ki, bu da yüksək və aşağı nəticə göstərən kateqoriyaların bir baxışda seçilməsini təmin edir.

### Əsas tapıntılar:
Analiz nəticəsində məlum olub ki, mağazanın illik gəlirinin əsas hissəsini pişik və it məhsulları təşkil edir. Xüsusilə, "Equipment" (Avadanlıq) və "Toys" (Oyuncaqlar) kateqoriyaları satışlarda liderlik edir. Digər tərəfdən, hamster və dovşan məhsulları üzrə satış göstəriciləri nisbətən aşağıdır, bu da növbəti dövrlərdə inventar strategiyasının yenidən nəzərdən keçirilməsi üçün əsas verə bilər.
